# PaRang Stikka Platinum [o_o]

Welcome to **PaRang Stikka Platinum**, the ultimate web-based factory for thermal sticker production. This project is a modern, feature-rich evolution designed to make sticker making fast, fun, and accessible to everyone.

(^_^)v

---

## What is this? [?]

PaRang Stikka Platinum is a browser-based application that allows you to design and print stickers, labels, memes, and even large posters using thermal printers. Whether you are using a professional label printer or a DIY setup, this tool bridges the gap between your screen and the physical world.

It is built with HTML, CSS, and JavaScript, requiring no complex installation. Just open the file and start creating! <( " )>

---

## Features [!]

- **Sticker Studio**: Upload any image and apply high-quality dithering for that classic thermal print look.
- **Label Maker**: Quick text labels with adjustable fonts, sizes, and optional QR codes.
- **Meme Machine**: Turn your favorite images into physical memes with top and bottom text.
- **Drawing Pad**: Draw directly in the browser with live dithered previews. [~_~]
- **Poster Puzzle**: Split a single large image into multiple panels for giant "poster" prints.
- **Webcam Snap**: Take a photo and print it instantly.
- **Random Generators**: Fetch random cats, dogs, or AI-generated people for quick sticker fun.
- **History Tracking**: Keep track of your past prints so you can re-print them easily.

- **MQTT**: Connect to a remote printer bridge (like a Raspberry Pi or ESP32).
- **USB**: Direct printing via browser-supported protocols.

## Quick Start [>]

1. **Clone or Download** this repository.
2. Open `parang-stikka.html` in any modern web browser.
3. Select your **Printer Settings** (the gear icon ⚙).
4. Choose a module (Sticker, Label, etc.).
5. Hit **PRINT** and watch the magic happen! \o/
